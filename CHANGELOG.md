# Changelog

All notable changes to the "JSON Tool Kit" extension will be documented in this file.

## [0.1.0] - 2026-06-29

### Added
- Initial release of the **JSON Tool Kit** extension.
- Support for **Escape JSON String** (`jsonStringTools.escape`).
- Support for **Unescape JSON String** (`jsonStringTools.unescape`).
- Support for **Stringify Selection** (`jsonStringTools.stringify`).
- Support for **Parse JSON String** (`jsonStringTools.parse`).
- Support for **Pretty Print JSON** (`jsonStringTools.pretty`).
- Support for **Minify JSON** (`jsonStringTools.minify`).
- Multi-cursor selections and full-document fallback support.
- Fully lightweight, zero runtime dependency architecture.
